Add a dismissible announcement / notification bar to the top of your site — no theme editing.

== Description ==

Announce a sale, a notice, or a link at the top of every page. Set your message,
colours and an optional call-to-action button, tick "Show", and you're done.
Visitors can dismiss it (remembered for 7 days).

**Free features**

* Custom message and call-to-action button.
* Background and text colours.
* Dismissible with 7-day memory.

**Pro features**

* Sticky bar that stays on scroll.
* Scheduling and multiple bars.
* Priority support.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate.
3. Go to Settings → Slim Announcement Bar, set your message, and tick "Show".

== Frequently Asked Questions ==

= Will it match my theme? =

You pick the background and text colours; the font is inherited from your theme.
